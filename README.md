# Quantum Ignition Demonstrator 🚀

下一代EUV光源技術的開源演示框架，展示量子預火光源的核心創新。

## ✨ 核心特性

- **量子噴流控制** - 納米級精確的量子膠囊生成
- **神曲自適應算法** - 智能多目標優化引擎  
- **ABN-QSS控制器** - 基於魔方陣的穩定控制架構
- **量子裁剪物理** - 高效光子能量轉換

## 🛠 快速開始

### 安裝
```bash
git clone https://github.com/hamishwork00a-alt/quantum-ignition-demonstrator.git
cd quantum-ignition-demonstrator
pip install -r requirements.txt
```

基礎演示

```python
from src.demonstrations.basic_demo import QuantumIgnitionDemo

demo = QuantumIgnitionDemo()
demo.run_full_demonstration()
```
